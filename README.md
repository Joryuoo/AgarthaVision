# AgarthaVision

> A digital information and diagnostic system for Soil-Transmitted Helminth (STH) surveillance.

## Overview

AgarthaVision is a mobile-based diagnostic support and surveillance platform for detecting and documenting Soil-Transmitted Helminths in fecal smear microscopy workflows. The system captures microscopy images, sends them for AI-assisted detection, computes Eggs Per Gram (EPG), and provides a Human-in-the-Loop (HITL) verification dashboard where medical technologists can review, correct, and validate results before they are used for reporting.

The system is designed as a decision-support and preprocessing tool. It does **not** replace the judgment of qualified medical personnel. Instead, it improves traceability, reduces manual scanning burden, and supports structured surveillance reporting.
