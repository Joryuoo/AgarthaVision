# 1.2 Image Payload Transmission

```mermaid
sequenceDiagram
autonumber

participant FS as FrameSampler
participant IFC as InferFrameUseCase
participant API as InferenceApi (Retrofit)
participant INF as Inference Container
participant FFS as FlaggedFrameStore
participant PFF as PersistFlaggedFrameUseCase
participant SDAO as SampleDao (Room)
participant SIS as SampleImageStore
participant DID as DeviceIdProvider
participant AUTH as AuthRepository

FS->>IFC: invoke(sessionId, jpegBytes)
IFC->>API: POST /infer (image/jpeg body)
API->>INF: HTTP request
INF-->>API: 200 + predictions[]
API-->>IFC: Response(InferenceResponseDto)
alt predictions is empty
  IFC-->>FS: return
else predictions found
  IFC->>FFS: add(FlaggedFrame)
  FFS->>PFF: invoke(frame)
  PFF->>AUTH: getCurrentUserId()
  AUTH-->>PFF: userId
  PFF->>SIS: persistJpeg(userId, sampleId, bytes)
  SIS-->>PFF: imagePath
  PFF->>DID: id
  DID-->>PFF: deviceId
  PFF->>SDAO: insertSample(SampleEntity)
  SDAO-->>PFF: ok
  PFF-->>FFS: sampleId
end
```

```mermaid
sequenceDiagram
autonumber

participant VM as Verification/Manual Submit (UseCase)
participant SDAO as SampleDao (Room)
participant DDAO as DetectionDao (Room)
participant LOC as LocationProvider
participant SYNC as SyncSampleUseCase
participant RDS as SampleRemoteDataSource
participant SB as Supabase (Storage + PostgREST)

VM->>LOC: getCurrentLocation()
LOC-->>VM: location?
VM->>SDAO: updateSampleOnVerify(...)
VM->>DDAO: insertDetections(...) / insertDetection(...)
VM->>SYNC: invoke(sampleId)
SYNC->>SDAO: getSampleById(sampleId)
SYNC->>DDAO: getDetectionsForSample(sampleId)
SYNC->>RDS: syncSample(sample, detections, imageBytes)
RDS->>SB: Storage.upload(userId/sampleId.jpg)
RDS->>SB: PostgREST.insert(samples row)
opt detections not empty
  RDS->>SB: PostgREST.insert(detections rows)
end
RDS-->>SYNC: storagePath
SYNC->>SDAO: updateSyncMetadata(status=SYNCED, storagePath)
```
