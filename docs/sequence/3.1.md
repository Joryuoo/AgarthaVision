# 3.1 Session-Based User Report Generation

```mermaid
sequenceDiagram
autonumber

participant UI as SessionDetailScreen (Compose)
participant VM as SessionDetailViewModel
participant OBS as ObserveSessionReportsUseCase
participant GEN as GenerateSessionReportUseCase
participant AUTH as AuthRepository
participant RRepo as ReportRepository (Room)

UI->>VM: screen start (sessionId from nav)
VM->>OBS: invoke(sessionId)
OBS->>AUTH: getCurrentUserId()
alt not logged in
  OBS-->>VM: emit []
else logged in
  OBS->>RRepo: observeForSession(sessionId, userId)
  RRepo-->>VM: Flow<List<Report>>
end
VM-->>UI: render reports list
```

```mermaid
sequenceDiagram
autonumber

participant User
participant UI as SessionDetailScreen (Compose)
participant VM as SessionDetailViewModel
participant GEN as GenerateSessionReportUseCase
participant AUTH as AuthRepository
participant SRepo as SessionRepository
participant SampRepo as SampleRepository
participant DRepo as DetectionRepository
participant CSV as ReportCsvBuilder
participant FS as ReportFileStore (DownloadsReportFileStore)
participant RRepo as ReportRepository (LocalReportRepository)
participant Sync as SyncReportUseCase
participant RDao as ReportDao
participant RDS as ReportRemoteDataSource
participant SB as Supabase (PostgREST)

User->>UI: tap Generate Report
UI->>VM: generateReport()
VM->>GEN: invoke(sessionId)
GEN->>AUTH: getCurrentUserId()
AUTH-->>GEN: userId
GEN->>SRepo: getSessionById(sessionId)
SRepo-->>GEN: session
GEN->>SampRepo: getSamplesForSession(sessionId, userId)
SampRepo-->>GEN: samples
GEN->>DRepo: getDetectionsForSample(sampleId) (per sample)
GEN->>DRepo: getConfirmedEggCountsForSession(sessionId, userId)
GEN->>CSV: build(...)
CSV-->>GEN: csv text
GEN->>FS: writeCsv(reportId, sessionId, csv)
FS-->>GEN: csvFilePath
GEN->>RRepo: insert(report{PENDING})
RRepo-->>GEN: ok
GEN->>Sync: invoke(reportId)
Sync->>RDao: getReportById(reportId)
RDao-->>Sync: reportEntity
Sync->>RDS: upsertReport(reportEntity)
RDS->>SB: insert into public.reports
SB-->>RDS: ok
RDS-->>Sync: ok
Sync->>RDao: updateSupabaseStatus(SYNCED)
Sync-->>GEN: Result
GEN-->>VM: Result<Report>
alt success
  VM-->>UI: emit ReportGenerated(csvFilePath)
else failure
  VM-->>UI: show error
end
```
