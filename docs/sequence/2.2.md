# 2.2 Human-in-the-Loop Validation

```mermaid
sequenceDiagram
autonumber

participant User
participant UI as VerificationSheet (Compose)
participant VM as VerificationViewModel
participant FFS as FlaggedFrameStore
participant SUB as SubmitVerificationUseCase
participant AUTH as AuthRepository
participant LOC as LocationProvider
participant SDAO as SampleDao (Room)
participant DDAO as DetectionDao (Room)
participant SYNC as SyncSampleUseCase

User->>UI: answer Q1/Q2/Q3/Q4 + optional note
UI->>VM: onQ1Selected/onQ2Selected/onSpeciesSelected/onQ4Selected/onUserNoteChanged
opt mark as repeat
  User->>UI: toggle Repeat
  UI->>VM: onToggleRepeat()
  VM->>FFS: toggleRepeat(frame)
end

User->>UI: Submit
UI->>VM: onSubmit()
VM->>SUB: invoke(frame, answers, missedEgg, userNote, isRepeat)
SUB->>AUTH: getCurrentUserId()
AUTH-->>SUB: userId
SUB->>LOC: getCurrentLocation()
LOC-->>SUB: location?
SUB->>SDAO: updateSampleOnVerify(...)
SUB->>DDAO: insertDetections(detections)
SUB->>SYNC: invoke(sampleId)
SYNC-->>SUB: Result
alt success
  VM->>FFS: remove(frame)
  VM-->>UI: emit Dismiss event
else failure
  VM-->>UI: emit ShowError(message)
end
```
