# 1.1 Live Image Acquisition

```mermaid
sequenceDiagram
autonumber
participant UI as MicroscopyViewport (Compose)
participant CM as CameraManager
participant CX as CameraX (Preview + ImageAnalysis)
participant FS as FrameSampler (ImageAnalysis.Analyzer)
participant SM as SessionManager

UI->>CM: bindAnalysis(lifecycleOwner, previewView, analyzer=FS)
CM->>CX: bindToLifecycle(Preview, ImageAnalysis.setAnalyzer(FS))
loop continuous frames
  CX-->>FS: analyze(imageProxy)
  FS->>FS: toJpegBytes()\n_latestFrameBytes = jpegBytes
  FS->>SM: read state (SessionState)
  alt SessionState.Active && isInferenceRunning
    FS->>FS: throttle (2s) + inFlight guard
    note over FS: When allowed, FrameSampler dispatches\ninference on a background coroutine.
  else Idle / paused
    FS-->>FS: return (no inference dispatch)
  end
end
```
