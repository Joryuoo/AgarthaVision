# 2.1 Diagnostic Result Rendering

```mermaid
sequenceDiagram
autonumber

participant IFC as InferFrameUseCase
participant FFS as FlaggedFrameStore
participant CVM as CaptureViewModel
participant UI as CaptureScreen (Compose)
participant SheetVM as VerificationViewModel
participant Sheet as VerificationSheet (Compose)
participant FWB as FrameWithBoxes (Compose)

IFC->>FFS: add(FlaggedFrame)
FFS-->>CVM: state emits frames (Flow/StateFlow)
CVM-->>UI: state update (flaggedFrames)
UI-->>UI: render toast/queue badge/etc.

opt user taps detection toast
  UI->>CVM: onDetectionToastTap(frame)
  CVM-->>UI: state.verificationTarget = frame
  UI->>SheetVM: setFrame(frame)
  UI->>Sheet: mount
  Sheet->>FWB: render(jpegBytes, predictions, highlightedIndex, showBoxes)
end
```
